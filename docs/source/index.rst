Welcome to the feiertage documentation!
=======================================

**feiertage** is a package that calculates all German national holidays for a given year and a given federal state. The national holidays will be assigned by the `table in the wikipedia article about german holidays <https://de.wikipedia.org/wiki/Gesetzliche_Feiertage_in_Deutschland#%C3%9Cbersicht_aller_gesetzlichen_Feiertage>`_ 

**feiertage** ist ein Python-Packet das die Feiertage für die Bundesländer der Bundesrepublik Deutschland berechnet.
